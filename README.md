# ShortAI — React/Vite App

AI alati za kreatore kratkih videa. Konvertovano iz standalone HTML u React + Vite za deployment na Lovable/Vercel/Netlify.

## 🚀 Pokretanje lokalno

```bash
npm install
npm run dev
```

## 🏗️ Build za produkciju

```bash
npm run build
```

## 📦 Stack

- **React 18** + **React Router v6**
- **Vite** build tool
- **Nema backend-a** — sve radi u browseru
- **IndexedDB** za lokalnu biblioteku videa
- **MediaRecorder API** za generisanje videa

## 🔑 API Postavke

Otvori ⚙️ → unesi API ključ:
- **Groq** (besplatno): [console.groq.com](https://console.groq.com)
- **Anthropic**: [console.anthropic.com](https://console.anthropic.com)
- **OpenAI**: [platform.openai.com](https://platform.openai.com)

## 🌐 Deploy na Lovable

1. Uploaduj ovaj folder na GitHub
2. Importuj repo u [lovable.dev](https://lovable.dev)
3. Lovable automatski deployjuje na `*.lovable.app` URL

## 📲 APK (Median.com)

1. Deploy na Lovable/Netlify/Vercel
2. Kopiraj URL (npr. `https://shortai.lovable.app`)
3. Unesi u [median.com](https://median.com) za Android APK
